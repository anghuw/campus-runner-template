# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-31

### Added
- Initial frontend template with Expo SDK 56 + React Native 0.85
- Bottom tab navigation with 5 tabs: Home, Orders, Publish, Messages, Profile
- Stack navigation for detail pages
- Home page with task list, search, and category filters
- Order list page with tab switching (published / accepted)
- Publish task page with form inputs
- Messages page with conversation list
- Profile page with user info and menu items
- Order detail page with status timeline
- Chat page with message bubbles
- Runner profile page
- Search page with keyword filtering
- Rating page with star rating and comment
- Zustand state management with mock data
- TypeScript type definitions
- Utility functions for formatting dates, prices, and distances
- MIT License

### Notes
- This release is a **frontend template with mock data only**
- No backend, no authentication, no real data persistence
- Designed as a starting point for campus errands app development
